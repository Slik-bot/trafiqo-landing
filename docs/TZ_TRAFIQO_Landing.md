## 2. ДИЗАЙН-КОНЦЕПЦИЯ

### 2.1 Настроение и вайб
Тёмный премиум. Интрига. Технологичность без перегруза.
Сайт должен говорить: *«Мы знаем что делаем — и сделаем это для вас».*
Клиент видит сайт → думает: *«Если их сайт так выглядит — моё тоже будет на уровне».*

### 2.2 Цветовая палитра

| Цвет | HEX | Назначение |
|---|---|---|
| Фон основной | `#07050F` | Базовый фон всего сайта |
| Фон свечение | `#1A0A3A` | Radial gradient в Hero |
| Фиолетовый акцент | `#5B2D8E` | Кнопки, иконки, подсветка |
| Фиолетовый светлый | `#9B7FE8` | Цифры, стрелки, hover |
| Текст основной | `#F0ECFF` | Заголовки |
| Текст вторичный | `rgba(232,228,240,0.5)` | Подзаголовки, описания |
| Текст мутный | `rgba(232,228,240,0.25)` | Мелкие подписи, копирайт |
| Граница | `rgba(255,255,255,0.08)` | Карточки, разделители |

### 2.3 Типографика

| Роль | Шрифт | Размер | Трекинг |
|---|---|---|---|
| Логотип / бренд | Space Grotesk или Syne | 14px | 5px |
| Заголовки H1 | Syne или Outfit | clamp(28px, 4vw, 48px) | -0.5px |
| Заголовки H2 | Syne | clamp(22px, 3vw, 36px) | -0.3px |
| Основной текст | Inter | 14–16px | 0 |
| Метки секций | Inter | 10px | 3px |

**Правило:** Максимум 2 шрифта на проект. Google Fonts. CDN.

### 2.4 Фон Hero — только CSS, без изображений

```css

  --bg-base: #07050F;

body {

```

Зернистость/текстура — через SVG-фильтр (`feTurbulence`), 3 строки, 0 KB.

### СЕКЦИЯ 1 — HERO (первый экран)

**Цель:** Захватить внимание. Передать масштаб. Призвать к действию.

**Контент:**
- Метка: `DIGITAL AGENCY` (10px, letter-spacing 3px, фиолетовый)
- H1: `Строим цифровые продукты, которые работают на ваш бизнес`
- Подзаголовок: `Сайты · Приложения · Автоматизация · Боты · Платформы`
- CTA primary: `Обсудить проект` (фиолетовая кнопка с glow)
- CTA secondary: `Смотреть направления` (прозрачная, с бордером)

**Визуал (HTML + CSS, без изображений):**
- Слева — текст + CTA
- Справа — три вырезанных UI-объекта:
  - Браузер с макетом сайта (наклон -3°)
  - Телефон с приложением (наклон +4°)
  - Чат-бот / окно автоматизации (наклон -1°)

**Анимации:**
- GSAP timeline при загрузке: логотип → метка → H1 → подзаголовок → CTA → UI-объекты (stagger 0.15s, fade + translateY)
- Параллакс: UI-объекты двигаются при скролле (GSAP ScrollTrigger, только transform)
- Десктоп: параллакс активен
- Мобайл: параллакс отключён, объекты стопкой под текстом

## 4. ПАРАЛЛАКС — ЧТО И КАК

### Объекты параллакса (всё HTML+CSS, без изображений):

| Объект | Слой | Скорость | Где |
|---|---|---|---|
| Фиолетовое glow (CSS radial-gradient) | 0 — самый дальний | 0.1× скорости скролла | Hero, весь сайт |
| Браузер с сайтом | 1 | 0.3× | Hero |
| Телефон с приложением | 2 | 0.5× | Hero |
| Чат-бот окно | 1.5 | 0.4× | Hero |
| Декоративные точки/линии | 0.5 | 0.2× | По всему сайту |

### Правило параллакса на мобайле:
```javascript
const isMobile = window.innerWidth < 768;
const prefersReduced = window.matchMedia('(prefers-reduced-motion: reduce)').matches;

if (!isMobile && !prefersReduced) {
  // инициализировать GSAP ScrollTrigger параллакс
}
```

На мобайле — параллакс полностью отключён. Только AOS-анимации.

## 8. АДАПТИВНОСТЬ

| Устройство | Ширина | Подход |
|---|---|---|
| Mobile | < 768px | Базовые стили (mobile-first) |
| Tablet | 768–1199px | @media (min-width: 768px) |
| Desktop | 1200px+ | @media (min-width: 1200px) |

### Правила адаптива:
- Шрифты: `clamp()` везде — никакого ручного переопределения
- Карточки услуг: горизонтальный скролл на мобайле
- Hero: колонки → стопка на мобайле
- Параллакс: отключён на мобайле
- Навигация: бургер-меню на мобайле
- Форма: поля на всю ширину
